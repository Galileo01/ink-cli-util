## CLI

```
Usage
   $ ink-cli-util
      scan  扫描模式
      diff  Diff模式

 Options
    scan 扫描模式
      --keyword,-k  搜索关键词 keyword 
      --dir,-d  搜索目录 dir 默认 './src'  
      --out-dir,-o  文件输出目录 out-dir 默认 './scan-output'


    diff Diff模式

 Examples
   $ ink-cli-util scan --keyword=useAccess
```
